Krestiki-noliki 1.0.0

Проект реализует игру в крестики нолики с заданными правилами,
которые указаны в правилах курсового проектирования.

## Установка
# Для Windows
Необходимо Visual Studio 2019
Открыть .sln в предыдущем каталоге, запустить.

# Для Xubuntu
Установить	apt install make
		apt install gcc
		apt install libgl1-mesa-dev

Прописать make в данной директории
